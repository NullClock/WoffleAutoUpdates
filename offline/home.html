keyboard shortcuts

Hover on the top bar to execute Woffle's keyboard shortcuts.

  • Alt + W to close current tab
  • Ctrl + Alt + W to erase this window's history
  • Alt + T to open new tab
  • Alt + N to open new window
  • Alt + E to view all Woffle tabs
  • Alt + D to toggle Discord
  • Alt + / to toggle on/off fog mode (blurs screen if inactive for more than 15 seconds or is in alt+tab switcher)
  • Alt + i to toggle between themes
  • Alt + - to minimize
  • Alt + + to resize
  • Alt + [ to multitask left
  • Alt + ] to multitask right
  • Alt + Esc to make top bar sticky/not sticky
